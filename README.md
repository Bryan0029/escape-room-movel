# Escape Room Móvel 

Landing page responsiva para startup fictícia de Escape Room — desenvolvida com HTML5 e CSS3 puros como projeto acadêmico no IFNMG Montes Claros.

## Sobre o projeto

Página de vendas para uma startup fictícia que leva experiências de Escape Room para empresas e escolas.

## Tecnologias utilizadas

- HTML5 semântico
- CSS3 com Flexbox
- Google Fonts (Rajdhani + Crimson Text)
- Font Awesome (ícones)

## Funcionalidades

- Banner Hero com imagem de fundo fixa (parallax)
- Catálogo de 8 salas com hover animado
- Tabela de preços oculta com details/summary
- Hall da Fama com ranking por sala
- Layout com aside fixo de 250px e calc()
- Header sticky
- Responsivo para tablet e mobile

## Como visualizar

Basta abrir o arquivo `index.html` no navegador — não requer servidor ou instalação.
